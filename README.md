# AI RAG and LLM Starter Project

This repository contains a simple Python-based AI starter project that demonstrates LLM API integrations, prompt-based workflows, and Retrieval-Augmented Generation (RAG) experiments using LangChain, ChromaDB, and OpenAI-compatible APIs.

## Features

- Example Python entry point for running a basic AI script
- Notebook-based exploration for LLM API calls
- RAG workflow setup with ChromaDB and document retrieval
- Easy dependency management through the project configuration file

## Getting Started

### Prerequisites

- Python 3.10 or newer
- pip
- Access to an OpenAI-compatible API key if you want to run the LLM and RAG examples

### Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd AI
```

2. Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Install the project dependencies:

```bash
pip install -e .
```

## Usage

Run the basic entry point:

```bash
python main.py
```

Explore the notebooks in the LLM and RAG folders to experiment with API calls and retrieval pipelines.
